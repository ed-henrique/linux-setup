# How To Setup Keyboard Shortcuts in KDE Plasma

| Trigger          | Action                      |
| :--------------: | :-------------------------: |
| `ALT + SPACE`    | Search                      |
| `CTRL + ALT + T` | /opt/Hyper                  |
| `COMMAND + D`    | Minimize all                |
| `PRINT`          | Take screenshot (Flameshot) |
